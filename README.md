# SafeGuard

**Plataforma de Educação em Cibersegurança**
SafeGuard é uma landing page institucional voltada para a conscientização e o ensino de práticas de cibersegurança para o público geral, abordando desde fundamentos de proteção digital até tópicos voltados a desenvolvedores.
🔗 **Acesse o projeto:** [hilliangabriel.github.io/SafeGuard](https://hilliangabriel.github.io/SafeGuard/)

---

## Sobre o Projeto
O projeto foi desenvolvido como avaliação prática da disciplina **Mentalidade Criativa e Empreendedora e Programação Front-End**, com o objetivo de apresentar, por meio de uma landing page moderna e responsiva, os principais pilares da cibersegurança: fundamentos, proteção de dispositivos, privacidade de dados, segurança de redes, engenharia social e segurança no desenvolvimento de software.

## Decisões de Design (UX/UI)
- **Dark Mode nativo** — reduz cansaço visual e transmite identidade tecnológica
- **Hierarquia visual e tipografia** com a fonte Inter
- **Whitespace** generoso para melhor processamento de informação
- **Layout em grid (Bootstrap) Mobile First** — responsivo em qualquer dispositivo

## Recursos Visuais
- **Glassmorphism** na barra de navegação (`backdrop-filter: blur`)
- **Glow Effects** (brilho neon) em cartões e elementos de destaque
- **Gradientes de texto** para realçar palavras-chave
- **Micro-interações** em hover (transições suaves e efeito de flutuação)

## Funcionalidades
- Componentização semântica com HTML5 (`nav`, `section`, `footer`)
- Animações baseadas em rolagem com **GSAP ScrollTrigger**
- Contadores dinâmicos orientados a dados (formatação `pt-BR`)
- Ícones vetoriais escaláveis com **Font Awesome 6**

## Tecnologias Utilizadas
|---|---|
| HTML5 | Estruturação semântica da página (nav, section, footer) |
| CSS3 | Estilização customizada (dark mode, glassmorphism, glow, gradientes), embutida no `index.html` |
| Bootstrap 5.3.8 | Sistema de grid responsivo e componentes |
| JavaScript | Animações e contadores, embutido no `index.html` |
| GSAP + ScrollTrigger | Animações de entrada conforme o scroll |
| Font Awesome 6 | Ícones vetoriais |
| GitHub Pages | Hospedagem da página estática |
